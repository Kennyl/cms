# Full-featured nodejs CMS (beta)

## Installation

```
download, unzip
```

```
cd inside folder, than "npm install" to download modules
```

```
run with "node debug.js" ,or "node delease.js"
```

## Requirements
- nodejs >= 0.11
- running mongodb >= 2

## Next Steps

2. click "sign up" and enter admin email/pass - this will create first admin user
3. log in
